## 🧠 Solution Highlights — My Accepted Solutions

### ✅ LC 234 · Palindrome Linked List — 0ms
```
Strategy: Fast-Slow to find mid → Reverse 2nd half → Compare both halves
Time: O(n)  |  Space: O(1)
```
```cpp
bool isPalindrome(ListNode* head) {
    ListNode *slow=head, *fast=head, *prev=NULL;
    while(fast && fast->next){ slow=slow->next; fast=fast->next->next; }
    while(slow){ ListNode* next=slow->next; slow->next=prev; prev=slow; slow=next; }
    ListNode *left=head, *right=prev;
    while(right){ if(left->val!=right->val) return false; left=left->next; right=right->next; }
    return true;
}
```

### ✅ LC 203 · Remove Linked List Elements — 0ms
```
Strategy: Dummy head node to avoid edge cases at head
Time: O(n)  |  Space: O(1)
```
```cpp
ListNode* removeElements(ListNode* head, int val) {
    ListNode* dummy = new ListNode(-1); dummy->next = head;
    ListNode* curr = dummy;
    while(curr->next){ if(curr->next->val==val) curr->next=curr->next->next; else curr=curr->next; }
    return dummy->next;
}
```

### ✅ LC 202 · Happy Number — 0ms
```
Strategy: Unhappy numbers always cycle through 4 → stop there
Time: O(log n)  |  Space: O(1)
```
```cpp
int squareSum(int n){ int s=0; while(n>0){int d=n%10; s+=d*d; n/=10;} return s; }
bool isHappy(int n){ while(n!=1 && n!=4) n=squareSum(n); return n==1; }
```

### ✅ LC 2095 · Delete the Middle Node — 0ms
```
Strategy: Fast pointer moves 2x speed → when done, slow is at mid
Time: O(n)  |  Space: O(1)
```
```cpp
ListNode* deleteMiddle(ListNode* head) {
    if(!head || !head->next) return NULL;
    ListNode *slow=head, *fast=head->next->next;
    while(fast && fast->next){ slow=slow->next; fast=fast->next->next; }
    slow->next=slow->next->next;
    return head;
}
```

### ✅ LC 171 · Excel Sheet Column Number — 0ms
```
Strategy: Base-26 positional system like decimal but letters
Time: O(n)  |  Space: O(1)
```
```cpp
int titleToNumber(string s){
    int ans=0;
    for(char c:s) ans=ans*26+(c-'A'+1);
    return ans;
}
```
